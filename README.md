# Machine_Learning_SKLearn


Primeiro treino e teste de um modelo de classificação 
Nesta aula, aprendemos sobre classificação de animais entre porcos e cachorros utilizando a linguagem Python. Criamos variáveis para representar cada animal, com um conjunto de valores que representam suas características. Utilizamos a biblioteca Scikit Learn para criar um modelo de aprendizado de máquina que pudesse estimar a classe de um animal desconhecido com base nas características fornecidas. O modelo utilizado foi o LinearSVC, que é capaz de realizar classificações lineares. Treinamos o modelo com os dados fornecidos e calculamos a taxa de acerto do modelo, que foi de 66.66%.
